# ­ЪїГ CXELI-DZAGLI-PICKER | рЃфрЃ«рЃћрЃџрЃў рЃФрЃљрЃдрЃџрЃўрЃА рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃарЃћрЃюрЃЊрЃЮрЃЏрЃљрЃўрЃќрЃћрЃарЃў

## ­ЪўФ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃЏрЃљрЃф DevOps-рЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃЦрЃЏрЃюрЃљ

рЃДрЃЋрЃћрЃџрЃљрЃА рЃњрЃЋрЃўрЃДрЃЋрЃљрЃарЃА рЃГрЃљрЃЏрЃљ рЃЊрЃљ рЃАрЃћрЃарЃўрЃљрЃџрЃўрЃА рЃДрЃБрЃарЃћрЃЉрЃљ. рЃЕрЃћрЃЏрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃАрЃљрЃБрЃЉрЃљрЃарЃўрЃљ рЃџрЃћрЃњрЃћрЃюрЃЊрЃљрЃарЃБрЃџ рЃЦрЃљрЃарЃЌрЃБрЃџ рЃАрЃћрЃарЃўрЃљрЃџрЃќрЃћ Рђћ **"рЃфрЃ«рЃћрЃџрЃў рЃФрЃљрЃдрЃџрЃў"**.

рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃћрЃарЃЌрЃў рЃбрЃарЃљрЃњрЃћрЃЊрЃўрЃљ:

> РЮЌ **рЃАрЃљрЃГрЃЏрЃџрЃўрЃА рЃГрЃљрЃЏрЃљрЃА рЃЋрЃљрЃюрЃЊрЃЮрЃЏрЃћрЃЉ 5 рЃгрЃБрЃЌрЃА, рЃ«рЃЮрЃџрЃЮ рЃАрЃћрЃарЃўрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃА Рђћ 45 рЃгрЃБрЃЌрЃА!**

рЃћрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃўрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮ, рЃарЃЮрЃЏ рЃЋрЃћрЃдрЃљрЃа рЃЋрЃўрЃбрЃљрЃюрЃЊрЃў рЃўрЃЏ рЃБрЃљрЃќрЃарЃЮ рЃЊрЃарЃЮрЃўрЃА рЃ«рЃљрЃарЃ»рЃЋрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃћрЃарЃўрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃЮрЃЊрЃљ.

**рЃћрЃА рЃљрЃарЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃГрЃљрЃЏрЃўрЃА рЃЊрЃарЃЮрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃћрЃЏрЃАрЃљрЃ«рЃБрЃарЃћрЃЉрЃљ.**

---

## РџЎ№ИЈ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃћрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ?

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃўрЃарЃЕрЃћрЃЋрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃАрЃћрЃарЃўрЃљрЃА 5 рЃАрЃћрЃќрЃЮрЃюрЃўрЃЊрЃљрЃю рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃ«рЃАрЃюрЃўрЃА рЃЏрЃљрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў. рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃљрЃарЃўрЃА рЃцрЃћрЃарЃљрЃЊрЃў (рЃЏрЃљрЃЊрЃџрЃЮрЃЉрЃљ `colorama`-рЃА) рЃЊрЃљ рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃАрЃћрЃќрЃЮрЃюрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃА (рЃЏрЃљрЃњ. `3`), рЃљрЃАрЃћрЃЋрЃћ рЃАрЃарЃБрЃџрЃљрЃЊ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃарЃћрЃЪрЃўрЃЏрЃА (`random`).

## ­Ъџђ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ (рЃАрЃ«рЃЋрЃўрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ)
